noise

void noise( strings $value )

Noise filter.

Apply a noise transformation to the image. Valid values are the following strings:

  • 'Uniform'
  • 'Gaussian'
  • 'Multiplicative'
  • 'Impulse'
  • 'Laplacian'
  • 'Poisson'
