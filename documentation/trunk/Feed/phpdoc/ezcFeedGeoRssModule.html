Feed::ezcFeedGeoRssModule

Class ezcFeedGeoRssModule

Support for the GeoRss module: data container, generator, parser.

Specifications: http://www.w3.org/2003/01/geo/.

Create example:

  1.  <?php
  2.  // $feed is an ezcFeed object
  3.  $item = $feed->add( 'item' );
  4.  $module = $item->addModule( 'GeoRss' );
  5.  $module->lat = 26.58;
  6.  $module->long = -97.83;
  7.  ?>

Parse example:

  1.  <?php
  2.  // $item is an ezcFeedEntryElement object
  3.  $lat = isset( $item->GeoRss->lat ) ? $item->GeoRss->lat->__toString() : null;
  4.  $long = isset( $item->GeoRss->long ) ? $item->GeoRss->long->__toString() : null;
  5.  ?>
