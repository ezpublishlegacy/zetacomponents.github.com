Mail::ezcMailPop3TransportOptions

Class ezcMailPop3TransportOptions

Class containing the options for POP3 transport.

The options from ezcMailTransportOptions are inherited.

Example of how to use POP3 transport options:

  1.  $options = new ezcMailPop3TransportOptions();
  2.  $options->ssl = true;
  3.  $options->timeout = 3;
  4.  $options->authenticationMethod = ezcMailPop3Transport::AUTH_APOP;
  5.  
  6.  $pop3 = new ezcMailPop3Transport( 'pop3.example.com', null, $options );
