DatabaseSchema::ezcDbSchemaReader

Interface ezcDbSchemaReader

This class provides the base interface for schema readers.

This interface is extended by both a specific interface for schema readers who read from a file (@link ezcDbSchemaFileReader) and one for readers that read from a database (@link ezcDbSchemaDbReader).
